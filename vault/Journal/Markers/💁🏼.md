title: VIPs

---
This marker signifies a person important to a project - a VIP - e.g. a stakeholder.

The alias ☝🏻 is used to derive the project lead from the journal log instead of the front matter in the project document. The line next to the marker should contain a single reference to a document (typically one representing a person). That document is the project lead. If the line has no internal links in it, then the line will be ignored. If it has multiple, the system use the first one.

The following keyboard snippets on my Mac expand to a link to this marker or one of its aliases:

| Shortcut | Output           |
| -------- | ---------------- |
| ,mv      | `[[💁🏼]] `      |
| ,vl      | `[[💁🏼\|☝🏻]] ` |
