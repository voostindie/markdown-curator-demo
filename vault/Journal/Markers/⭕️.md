This marker signifies "current status". I use it for projects to log status updates in my daily log when I'm reviewing projects.

Note the `group-by-date` setting in the front matter! It ensures that `marked` query doesn't just output status entries line by line without, but does so sorted and grouped by date.

The aliases 🟢, 🟠, 🔴, ✅, 🗑️ are used to derive the project status from the journal instead of the front matter in the project document. The aliases themselves *are* the status.

The following keyboard snippets on my Mac expand to a link to this marker or one of its aliases:

| Shortcut | Output         |
| -------- | -------------- |
| ,ms      | `[[⭕️]] `      |
| ,sg      | `[[⭕️\|🟢]] `  |
| ,sa      | `[[⭕️\|🟠]] `  |
| ,sr      | `[[⭕️\|🔴]] `  |
| ,sd      | `[[⭕️\|✅]] `   |
| ,st      | `[[⭕️\|🗑️]] ` |
